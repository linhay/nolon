# Nolon v0.0.8 Release Notes

## 智能工作流与交互优化

本版本引入了全新的 Agent Workflows 概念，并进一步打磨了更新检测机制。

- **新特性: Agent Workflows**:
  - 正式引入了技能工作流（Workflow）的多维度管理。
  - 优化了工作流的状态流转逻辑，支持“纯净状态 (Clean State)”的一键恢复与管理。
- **发布与更新保障**:
  - **修复精细化**: 修复了之前版本中偶发的版本匹配逻辑错误，确保用户能准确收到后续更新推送。
  - **交互改进**: 进一步优化了 `McpServerCard` 及其环境变量配置面板的显示效果。
- **性能与稳定性**:
  - 针对大型 Skill 仓库的解析算法进行了优化，减少了长时间加载导致的 UI 阻塞。
  - 强化了 Deep Link 触发时的异常处理能力。
