# Nolon v0.1.0 Release Notes

## 自动化发布新篇章

作为一个里程碑版本，v0.1.0 标志着 nolon 的自动化发布体系正式成熟，并开始全面采用 Markdown 作为更新日志的源。

- **✨ 智能发布流程**:
  - **自动化提取**: 发布脚本现已支持自动从 `docs/RELEASE_NOTES_{version}.md` 读取内容，彻底告别手动同步。
  - **实时状态反馈**: 显著优化了发布过程中的 Log 输出，并在终端提供更直观的进度展示。
- **🐞 Bug 修复与精细化改进**:
  - **日志展示修复**: 彻底解决了 Sparkle 自动更新面板中偶尔出现的日志描述为空的问题。
  - **换行符转义**: 修复了发布至 GitHub Release 时，Markdown 内容由于换行符转义导致的格式错乱问题。
- **体验提升**:
  - 优化了 `AppCommandState` 状态管理，应用启动与各个面板之间的切换更加丝滑。
