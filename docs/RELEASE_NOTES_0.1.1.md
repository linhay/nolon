# Nolon v0.1.1 Release Notes

## 核心改进与修复

- **Skill 管理优化**: 
  - 修复了 `SkillCardView` 在特定场景下 Workflow 按钮点击无效的问题，现已支持点击直接切换关联状态。
  - 统一了 `McpServerCard` 与 `SkillCardView` 的 UI 结构，采用自上而下的规范布局（标题、菜单、版本、描述、操作区）。
  - 增强了卡片的可交互性，支持全卡片区域右键弹出操作菜单。
  
- **UI/UX 增强**:
  - 操作区按钮现在采用宽度自适应布局，避免强行平分布局导致的空间浪费。
  - 优化了启用、禁用及安装状态的背景色和不透明度区分，视觉反馈更明确。
  
- **本地化规范**:
  - 规范了中文本地化，确保 "MCP"、"Skill"、"Workflow" 等关键技术术语在翻译中保持英文原形，避免歧义。
  
- **系统稳定性**:
  - 修复了若干可能导致编译超时或顺序错误的 Swift 语法细节。
  - 验证了全项目的构建流程，确保在 Release 配置下的稳定性。
