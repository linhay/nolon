# Nolon v0.0.9 Release Notes

## 维护性更新与环境适应性

本版本是一个聚焦于稳定性的维护更新，重点解决了环境适配与细节交互问题。

- **环境适配优化**:
  - 优化了 MCP 服务器配置中环境变量的展开与过滤逻辑，支持更复杂的系统配置场景。
  - 增强了应用对不同区域设置（Locale）下文件路径的处理兼容性。
- **UI 微调**:
  - 精细调整了 `SkillCardView` 在网格模式下的间距与圆角，视觉感受更加精致。
  - 修正了部分多语言翻译中的技术术语使用。
- **持续集成**:
  - 进一步验证了在 macOS 15+ 环境下的代码签名与公证流程。
